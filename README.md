# Financial Health Dashboard

## Overview
This project is an interactive Financial Health Dashboard developed using Power BI. It provides key financial insights through interactive visualizations and KPI tracking.

## Key Metrics
- Revenue Analysis
- Expense Tracking
- Profit Monitoring
- Cash Flow Analysis
- Assets vs Liabilities Comparison
- Financial Performance Monitoring

## Tools Used
- Power BI
- Excel
- Data Visualization
